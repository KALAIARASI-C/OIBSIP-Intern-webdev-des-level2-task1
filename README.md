# Simple Calculator Web Application

## Objective

The objective of this project is to design and develop a simple web-based calculator that can perform basic arithmetic operations such as addition, subtraction, multiplication, and division. The project helps in understanding the fundamentals of front-end web development using HTML, CSS, and JavaScript.

## Steps Performed

1. Designed the basic structure of the calculator using **HTML**.
2. Created an input display area to show numbers and results.
3. Added buttons for digits (0–9), operators (+, -, *, /), decimal point, clear, delete, and equals.
4. Styled the calculator interface using **CSS** to create a modern dark-themed design with hover and click effects.
5. Implemented calculator logic using **JavaScript** functions such as:

   * `appendValue()` to add numbers or operators
   * `clearDisplay()` to clear the screen
   * `deleteLast()` to remove the last entered value
   * `calculate()` to compute the final result
6. Tested the calculator in a web browser to ensure correct functionality.
7. Uploaded the project files to **GitHub** for version control and project sharing.

## Tools and Technologies Used

* HTML5
* CSS3
* JavaScript
* Visual Studio Code (Code Editor)
* GitHub (Project Hosting)

## Outcome

The final outcome of this project is a fully functional **web-based calculator** with a user-friendly interface. The calculator allows users to perform basic mathematical calculations easily. The project demonstrates the integration of HTML for structure, CSS for styling, and JavaScript for functionality in building an interactive web application.
